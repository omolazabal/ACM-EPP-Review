Writing and Reading Objects

Applying what we learned above, we will read and write objects into and out of a file.

There are a few differences when dealing with objects. When you plan on writing an object to a file, you need to specify you will be appending to the file by including the ios::app flag in the open method. When writing, you will pass two parameters to the read method: a character pointer to your object and the size of the object.
